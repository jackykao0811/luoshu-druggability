## Overview

A parameter-free, sub-second protein druggability predictor using only backbone CA coordinates.

Encodes phi-psi dihedral geometry via the Luoshu (River Map) number framework into an 8-dimensional discrete conformational space, then derives convex-hull-based pocket descriptors.

### Key Results (n=123 proteins, Drug=87, Undrug=36)

| Metric | Value | 95% CI |
|--------|-------|--------|
| ROC-AUC | 0.861 | [0.778, 0.930] |
| PR-AUC | 0.928 | [0.868, 0.974] |
| F1 | 0.874 | — |
| LOPO-ROC | 0.861 | [0.778, 0.930] |
| LOPO-PR | 0.928 | [0.868, 0.974] |

**Head-to-head vs fpocket-inspired CA-only baseline (same data):**

|--------|---|--------|---------|
| ΔROC-AUC | +0.084 | [0.011, 0.164] | DeLong p=0.031 |
| ΔPR-AUC | +0.068 | [0.010, 0.134] | bootstrap p=0.007 |

## Method: Luoshu LTE Encoding

Each residue's phi-psi dihedral pair is mapped to a Luoshu trigram:

```
b1 = (|phi| <= 90)   # C-alpha in beta-sheet-like region
b2 = (phi > 0)       # dihedral orientation
b3 = (|psi| <= 60)   # helix-like compaction
```

This yields a Luoshu number L ∈ {1,2,3,4,6,7,8,9} (the 3×3 Luoshu magic square values).

The Local Transition Energy (LTE) at residue i is computed as:

```
LTE(i) = Σ_w φ^(-w) × Σ_{j≠i, |j-i|≤w} |L(i) - L(j)| / |i-j|
```

where φ = 1.6180 (golden ratio) and windows w ∈ {1, 3, 5, 7}.

Top-30th-percentile LTE residues define the putative binding pocket.
Their convex hull gives:
- **Sphericity** S = (36π V²)^(1/3) / A
- **CHC_max**: maximum depth from interior CA to hull boundary
- **Hydrophobic fraction** f_h

**Best model:** score = 0.60 × (S·CHC·f_h) + 0.25 × S + 0.15 × (S·f_h)

## Data

### Protein List (`data/protein_list.csv`)

| Column | Description |
|--------|-------------|
| `name` | Target name |
| `pdb_id` | RCSB PDB accession |
| `druggable` | 1 = druggable, 0 = undruggable |
| `family` | Structural family (Kinase, GTPase, etc.) |
| `source` | DUD-E, extended, or undrug-lit |

Labels: DUD-E benchmark annotations + established pharmacological classification (Hopkins & Groom 2002).

### Feature CSV (`data/luoshu_features.csv`)

Pre-computed features for 123 proteins (5 excluded for < 15 backbone residues):

| Column | Description |
|--------|-------------|
| `sphericity` | Pocket convex hull sphericity S |
| `chc_max` | Max convex hull concavity depth |
| `hydro_frac` | Fraction of hydrophobic residues in pocket |
| `sph_chc_hf` | Trilinear: S × CHC × f_h (Geometric Resonance feature) |
| `idim` | Intrinsic dimensionality (3=spherical, 2=flat) |
| `idim_chc` | Topological × depth product |
| `fp_vol` | Pocket volume (fpocket-style) |
| `fp_hydro` | Hydrophobic fraction (fpocket-style) |

## Key Findings

### 1. Geometric Resonance Hypothesis
The trilinear feature `sph × CHC × f_h` achieves ROC-AUC = 0.861 as a **single, parameter-free descriptor**. Druggability is a multiplicative property: roundness AND depth AND hydrophobicity must coexist. Any linear combination of the three terms allows compensatory trade-offs that degrade performance.

### 2. Intrinsic Dimensionality
Druggable pockets have `idim` ≈ 2.5–3.0 (genuine 3D cavities).  
Undruggable GTPases/PPI surfaces have `idim` ≈ 2.0 (degenerate 2D).  
The product `idim × CHC` achieves standalone ROC-AUC = 0.805.

### 3. LOPO = Full-Sample
LOPO cross-validation yields identical ROC-AUC (0.861) and PR-AUC (0.928), consistent with the parameter-free nature of the framework.
